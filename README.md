# create-serverless-nestjs-rest-api

Nest js rest api on aws lambda with serverless framework

## Installation

```bash
npx degit https://github.com/TheSmartMonkey/create-serverless-nestjs-rest-api.git nestjs-service
```

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run watch

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
